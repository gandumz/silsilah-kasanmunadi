⚙️ Hubungkan ke Google Sheets

Ikuti langkah berikut untuk menghubungkan silsilah ke Google Sheets agar data tersimpan online dan bisa diedit bersama:

  1. Buka sheets.google.com → buat spreadsheet baru
  2. Klik menu Extensions → Apps Script
  3. Hapus kode yang ada, lalu paste kode Apps Script di bawah ini
  4. Klik Deploy → New deployment → Web app
  5. Set Execute as: Me dan Who has access: Anyone
  6. Klik Deploy, izinkan akses, lalu copy Web app URL
  7. Paste URL tersebut di kolom di bawah ini
